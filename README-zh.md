# 烟花与无人机表演模拟器 (Shell Drone Animation)

[English](README.md) | [Tiếng Việt](README-vn.md) | [日本語](README-ja.md) | [简体中文](README-zh.md)

**Shell Drone Animation** 是一款交互式 3D 网页应用程序，允许用户编排和模拟烟花与无人机灯光秀。

## 主要功能

- **3D 模拟：** 开放世界模拟，并集成了编排工具包。
- **时间线编辑器 (Timeline Editor)：** 支持编辑、分享和导入剧本 JSON 文件的工具包。
- **3D 静态无人机编队 (3D Static Drone Formation)：** 设计并塑造无人机编队，内置导入现有 2D 和 3D 编队的支持及其他辅助工具。
- **无人机剧本编辑器 (3D Timeline Drone Editor)：** 按组编辑每个无人机编队的动画，并将其细分为逐步动画。
- **JSON 分享：** 导出 JSON 配置，以便在不同设备间轻松分享。

---

### 谁适合使用这款应用？

- 热衷于编排与音乐同步的烟花表演的爱好者。
- 需要设计和分享无人机编队矢量的用户。
- 想要为无人机创建逐步动画的任何人。
- 希望直接在电脑上观看烟花和无人机表演的人。

### 为什么选择这款应用？

- 内置专为编排和设计量身定制的丰富工具。
- 内置多种逼真度高达 80% 的烟花特效。
- 轻量且便捷的分享：导入即可直接运行。
- 逼真的画面提供身临其境的表演体验。

## 下载与安装

4. 打开下载的文件进行安装或直接运行软件。
