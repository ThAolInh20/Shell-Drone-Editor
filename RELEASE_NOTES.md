# Release Notes - v2.1.0

## Features
- Optimized firework explosion bursts by batching all active particles into a single global points system.
- Implemented instanced shell rendering to batch comet cores and rising shells into two draw calls.
- Optimized the smoke system by batching puffs into a single points system and reducing trail smoke emission rates.
- Updated the sky light reaction system effect.
- Added multi-language README documentations (English, Japanese, Vietnamese, Chinese) and release notes utility skill.
