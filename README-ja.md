# 花火＆ドローンショーシミュレーション (Shell Drone Animation)

[English](README.md) | [Tiếng Việt](README-vn.md) | [日本語](README-ja.md) | [简体中文](README-zh.md)

**Shell Drone Animation**は、ユーザーが花火とドローンのライトショーを演出し、シミュレーションできるインタラクティブな3Dウェブアプリケーションです。

## 主な機能

- **3Dシミュレーション:** 演出ツールキットを統合したオープンワールドシミュレーション。
- **タイムラインエディター (Timeline Editor):** シナリオJSONファイルの編集、共有、インポートをサポートするツールキット。
- **3D静的ドローンフォーメーション (3D Static Drone Formation):** ドローンフォーメーションのデザインと形成。既存の2Dおよび3Dフォーメーションのインポート機能と各種支援ツールを内蔵。
- **ドローンシナリオエディター (3D Timeline Drone Editor):** 各ドローンフォーメーションのアニメーションをグループごとに編集し、ステップバイステップのアニメーションに分割。
- **JSON共有:** JSON設定をエクスポートし、異なるPC間で簡単に共有。

---

### このアプリはどのような人向けですか？

- 音楽に合わせた花火の演出・振付に情熱を持つ方。
- ドローンのフォーメーションベクトルを設計し、共有したい方。
- ドローンのステップバイステップアニメーションを作成したい方。
- PCで直接、美しい花火やドローンショーを鑑賞したい方。

### なぜこのアプリを選ぶべきなのか？

- 演出やデザインに特化した豊富なツールを内蔵。
- 実物に最大80%忠実なリアルな花火エフェクトを多数内蔵。
- 軽量で簡単な共有：インポートするだけですぐに実行可能。
- リアルなグラフィックによる没入感のあるショー体験。

## ダウンロードとインストール

1. プロジェクトの**[GitHub Releases](https://github.com/ThAolInh20/Shell-Drone_3d/releases)**ページにアクセスします。
2. 最新のリリース（タグ）を選択し、Windows用のインストーラーをダウンロードします。
3. `.zip`または`.exe`ファイルをダウンロードします。
4. ダウンロードしたファイルを開き、インストールするか直接ソフトウェアを実行します。

*注意: インストール中にWindows SmartScreenのセキュリティ警告が表示される場合があります。「詳細情報 (More Info)」をクリックして「実行 (Run anyway)」を選択し、進めてください。*

本ソフトウェアは無料のオープンソースソフトウェアであり、ソースコードは**[Dev](https://github.com/ThAolInh20/Shell-Drone-Editor/tree/dev)**ブランチで直接確認できます。
