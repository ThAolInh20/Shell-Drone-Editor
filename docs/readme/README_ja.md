# 3D花火＆ドローンアニメーションシミュレーション (Shell Drone Animation)

**Languages:** [English](README_en.md) | [Tiếng Việt](README_vi.md) | [日本語](README_ja.md) | [中文](README_zh.md)

このプロジェクトは、**Three.js** と **Vite** を使用して構築され、**Electron** を利用してプロフェッショナルなデスクトップソフトウェアを構築した3D花火およびドローンアニメーションのシミュレーションおよび編集アプリケーションです。

---

## ダウンロードとインストールガイド

### 方法 1: Git を使用する (推奨)
Gitがインストールされている場合は、ターミナル（またはコマンドプロンプト/PowerShell）を開き、次のコマンドを実行します。
```bash
git clone https://github.com/ThAolInh20/Shell-Drone-Editor.git
cd shell-drone-animation
```

### 方法 2: ZIP ファイルを直接ダウンロードする
1. このリポジトリページの右上にある緑色の **Code** ボタンをクリックします。
2. **Download ZIP** を選択します。
3. ダウンロードしたZIPファイルを解凍し、ソースコードフォルダを開きます。

---

## 環境構築とアプリケーションの実行

このプロジェクトには **Node.js** バージョン 18 以上が必要です。先に進む前に Node.js がインストールされていることを確認してください。

### ステップ 1: 依存関係のインストール
プロジェクトのルートフォルダ (`shell-drone-animation`) でターミナルを開き、以下を実行します。
```bash
npm install
```

### ステップ 2: アプリケーションの起動

ニーズに応じて、2つの異なる方法でアプリケーションを実行できます。

#### 方法 A: Electron でデスクトップアプリとして実行 (推奨)
この方法は、Electronを使用して独立したデスクトップアプリケーションを起動します。この方法で実行すると、高度なネイティブ機能がロック解除されます。
*   **直接ファイル保存 (`Shift + S`):** ブラウザのダウンロードダイアログを経由**せず**に、編集内容をソースの `.json` ファイルに直接即座に保存します（プロのメモ帳のようなシームレスな体験）。
*   **クイックナビゲーションメニュー:** アプリケーションのシステムメニューまたはショートカット（`Ctrl + 1`、`Ctrl + 2`、`Ctrl + 3`）を介して、3つの編集ツールセット間をすばやく切り替えます。

デスクトップモードでアプリケーションを実行するには、次のコマンドを使用します。
```bash
npm run electron:dev
```

#### 方法 B: 従来の Web ブラウザで実行する
次を使用して標準の Vite 開発サーバーを実行します。
```bash
npm run dev
```
次に、ターミナルに表示されるローカルアドレス（デフォルトは `http://localhost:5173/`）にアクセスします。
*注意: Webブラウザで実行する場合、ブラウザのセキュリティメカニズムにより、直接ファイル保存機能はブラウザ経由のファイルダウンロードに自動的に切り替わります。*

## プロダクションビルド

### B. デスクトップアプリケーションのパッケージ化 (Electron による .EXE)
プロジェクト全体をWindows用の独立したデスクトップインストーラーソフトウェア（`.exe`）にパッケージ化する場合：

1. **ビルドコマンドの実行:**
   以下の単一のコマンドを実行して、Vite 経由でソースコードを自動的にコンパイルし、`electron-builder` でパッケージ化します。
   ```bash
   npm run electron:build
   ```
   生成されたファイルは `dist-electron` フォルダに保存されます。
